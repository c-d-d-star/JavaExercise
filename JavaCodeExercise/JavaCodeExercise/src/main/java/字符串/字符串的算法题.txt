1.判断回文字符串：判断一个字符串是否是回文字符串，即正读和反读都相同。


2.字符串反转：将一个字符串反转，即逆序输出字符串中的字符。



3.字符串拼接：将多个字符串按照一定的顺序拼接在一起。
方式一：
String str1 = "Hello";
String str2 = "World";
String result = str1 + " " + str2;
System.out.println(result);  // 输出: Hello World

方式二：
String str1 = "Hello";
String str2 = "World";
String result = str1.concat(" ").concat(str2);
System.out.println(result);  // 输出: Hello World


方式三：
String str1 = "Hello";
String str2 = "World";
// 使用 StringBuilder
StringBuilder sb = new StringBuilder();
sb.append(str1).append(" ").append(str2);
String result = sb.toString();
System.out.println(result);  // 输出: Hello World

// 使用 StringBuffer（线程安全）
StringBuffer buffer = new StringBuffer();
buffer.append(str1).append(" ").append(str2);
String result = buffer.toString();
System.out.println(result);  // 输出: Hello World

7.最长公共前缀：找出多个字符串中的最长公共前缀。



8.字符串解码：将编码后的字符串解码成原始字符串。



9.字符串压缩：对字符串进行压缩，将连续出现的字符进行计数压缩。
